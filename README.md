# 🩺 MediConnect - Online Consultation Platform

A simple and modern **frontend web application** for online medical consultation where users can book appointments, chat with doctors (UI simulation), and give reviews.

---

## 🌐 Live Pages
- Login Page
- Dashboard
- Book Appointment Page
- Doctor Chat Page
- Review Page
- Profile Page

---

## ✨ Features

✔ Clean **white and blue medical UI design**  
✔ Fully **multi-page website structure**  
✔ Login page with user name storage  
✔ Dashboard with navigation options  
✔ Book appointment form  
✔ Doctor chat system (frontend simulation with auto replies)  
✔ Review submission page  
✔ Profile page  
✔ Centered modern UI layout  
✔ Beginner-friendly project structure  

---

## 🧠 Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage (for username handling)
